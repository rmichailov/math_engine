# Math Engine

A C++ library for core mathematical algorithms, including **number theory**, **linear algebra**, **probability**, and **graph algorithms**. Designed for educational purposes, computational experiments, and as a foundation for more advanced math applications.

---

## Features

### Number Theory
- Greatest common divisor (GCD) and least common multiple (LCM)  
- Extended Euclidean algorithm (Bezout coefficients)  
- Primality tests and factorization (`prime_factors`, `prime_factorization`)  
- Divisors, totients, and coprimality checks  
- Perfect square checks
- Modular arithmetic (`mod_add`, `mod_mul`, `mod_pow`, `mod_inverse`)  
- Combinations and permutations (with optional modulo)  
- Chinese Remainder Theorem  
- Mobius and Jacobi functions  

### Linear Algebra
- Vector operations: addition, subtraction, scaling, dot product, norm, normalization  
- Matrix operations: addition, subtraction, scaling, multiplication, transpose, determinant, inverse, RREF  
- Matrix-vector multiplication

### Probability
- Discrete distributions: Bernoulli, Binomial, Geometric  
- Expected value computation  
- Monte Carlo integration and simulation  

### Tests
-Test Number Theory, Linear Algebra, and Probability methods
